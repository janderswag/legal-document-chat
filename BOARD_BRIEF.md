# BOARD_BRIEF

Drafted by the Chair from the founder's own words and a project scan, 2026-07-12. Correct anything wrong and the board will update it.

1. **What we're building:** DocuChat — a free, private, local-first legal document chat desktop app for attorneys. Documents never leave the machine; an open-source model runs locally. Signed and notarized macOS app, distributed via docuchat.app.

2. **Stage:** v0.6.0 published as Latest (2026-07-12). Golden gate 63/63. G-SCOPE verification layer shipped. Email connector live; a long list of connectors shown as "coming soon." Zero real attorney users so far.

3. **#1 goal right now:** Get real attorneys using it and giving feedback. Product stays free until attorneys are actually using it — no selling before that.

4. **#1 constraint:** Solo founder with a build-capability window — Fable 5 access runs through July 18, so maximum leverage is now. Windows code-signing is still unsolved. No revenue by design.
